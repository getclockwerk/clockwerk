# Clockwerk

AI-native time tracking for developers. Automatically tracks development time via AI tool hooks (Claude Code, Cursor, Codex, etc.).

## Install

```sh
curl -fsSL https://getclockwerk.com/install.sh | sh
```

## Quick start

```sh
clockwerk login
clockwerk init <project-token>
clockwerk up
```
